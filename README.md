# mithai-world

Production-ready MERN ecommerce starter with separate frontend and backend apps.

## Tech Stack
- MongoDB
- Express (ES modules)
- React (Vite)
- Node.js

## Project Structure
- backend/: Express API, auth, sockets, payment and delivery foundations
- frontend/: React app with routing and service layer

## Quick Start
1. Backend:
   - cd backend
   - npm install
   - Copy .env.example to .env and update values
   - npm run dev
2. Frontend:
   - cd frontend
   - npm install
   - npm run dev
